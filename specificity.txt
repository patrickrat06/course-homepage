Paragraph A: 
    Properties: color: gray, font-size: 1rem, font-family: Georgia.
    Receives its properties from the element selector (p).

Paragraph B: 
    Properties from .info: color: steelblue, font-size: 1.25rem, font-family: Arial.Inherited from p: line-height: 1.4, padding: 4px 0.
    Matches both p and the class selector (.info).

Paragraph C: 
    Properties from #warning: color: darkred, font-style: italic, letter-spacing: 0.15em.Inherited from p: font-family: Georgia, line-height: 1.4.
    Matches p and the id selector (#warning).

Paragraph D:
    Propeties from #warning.info: color: darkorange, font-size: 1.1rem, border: 2px solid darkorange.
    Properties from p: font-family: Georgia, line-height: 1.4.
    Matches p, .info, #warning, and the combined selector #warning.info.

2.

Color used by browser -> darkorange
This comes from the #warning.info selector. In CSS specificity, an ID + Class combination is more powerful than just an ID, Class or Element.

Font used by browser -> Georgia, serif
Because none of the stronger selectors contain a specification for the font, the browser still uses the one provided in (p), even though the other selectors are higher up in the hierarchy.
